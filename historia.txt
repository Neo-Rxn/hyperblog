Si quieres ver los archivos ocultos de una carpeta puedes habilitar la opción
 de Vista > Mostrar u ocultar > Elementos ocultos (en Windows) o ejecutar el 
 comando ls -a. o usando ls -al

 Le indicaremos a Git que queremos crear un nuevo repositorio para utilizar 
 su sistema de control de versiones. Solo debemos  posicionarnos en la carpeta raíz 
 de nuestro proyecto y ejecutar el comando git init.

 Recuerda que al ejecutar este comando (y de aquí en adelante) vamos a tener una nueva
 carpeta oculta llamada .git con toda la base de datos con cambios atómicos en nuestro
 proyecto. Nota si por error se elimina esta carpeta del S.O se perdera todo el repositorio
 y solo quedarian los archivos dentro de las carpetas, para recuperarlo tendriamo que ejecutar
 el comando "git init" de nuevo y se crearia un nuevo repositorio.
